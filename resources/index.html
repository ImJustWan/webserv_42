    About The Project

    We embarked on this project with the realization that our understanding of the workings of the web, particularly the HTTP protocol, was quite limited. Consequently, we began by delving into the process of how clients send requests to servers and how servers respond. This involved grasping the structure of messages (both requests and responses) and the underlying system calls (EPOLL) involved in this exchange, akin to a game of 'ping pong'. Additionally, we needed to familiarize ourselves with server configuration. Drawing inspiration from Nginx, we structured our configuration file and standards accordingly. In terms of implementing CGI, we opted to create both a form and a file uploader to enhance functionality.

    This website offers you the ability to accomplish two things:
    filling out a form and uploading a file.
